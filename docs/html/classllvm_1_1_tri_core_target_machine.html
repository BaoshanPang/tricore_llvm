Creates a TriCore machine archietecture.

The datalayout is described as below: e-m:e-p:32:32-i1:8:32-i8:8:32-i16:16:32-i64:64-f32:32-f64:64-a:0:32-n32-S64"

Meaning of symbols:

Symbol Definition
e little endian
p:32:32 pointer size: pointer ABI alignment
i1:8:32 integer data type : size of type : ABI alignment
f:64:64 float data type : size of type : ABI alignment
n32 all 32 bit registers are available
S64 atleasr 64 bits are available for the stack
